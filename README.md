# ECG Heart Rate Monitor 💓

Real-time ECG monitoring system using AD8232 sensor and Arduino.

## 🔧 Components Used
- Arduino Uno
- AD8232 ECG Sensor
- Electrodes (3 leads)
- Jumper Wires

## ⚡ How It Works
- AD8232 sensor captures heart electrical signals
- Arduino reads analog values from sensor
- Leads-off detection using pins 10 and 11
- Real-time data shown on Serial Monitor

## 📌 Pin Connections
- LO+ → Pin 10
- LO- → Pin 11
- Output → A0

## 🚀 How to Use
1. Connect AD8232 sensor to Arduino
2. Upload the code
3. Open Serial Monitor at 9600 baud
4. Attach electrodes and see ECG signal
